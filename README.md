###react-native学习之View组件
在React Native里有一个类似于div的组件，那就是View组件，支持多层嵌套，支持flexbox布局

实例步骤：

1.加载View组件

2.创建组件

3.添加样式表

4.注册入口

5.外层布局
```
<View style={styles.container}>
    <View style={styles.item}></View>
    <View style={styles.item}></View>
    <View style={styles.item}></View>
</View>
```
